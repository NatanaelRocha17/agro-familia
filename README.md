# 🌱 AgroFamilia

Vitrine para agricultores familiares, permitindo a divulgação dos seus produtos.

---

## 🚀 Tecnologias utilizadas

### Frontend

* React
* Vite
* TypeScript
* Tailwind CSS

### Backend

* Node.js
* Express
* MySQL
* JWT (autenticação)

---

## 📂 Estrutura do projeto

```
AgroFamilia/
├── frontend/
└── backend/
```

---

## ⚙️ Como rodar o projeto

### 🔹 Backend

```bash
cd backend
npm install
npm run dev
```

---

### 🔹 Frontend

```bash
cd frontend
npm install
npm run dev
```

---

## 🔐 Funcionalidades

* Cadastro de usuários
* Login com autenticação JWT
* Refresh Token
* Atualização de dados do usuário
* Integração com API
